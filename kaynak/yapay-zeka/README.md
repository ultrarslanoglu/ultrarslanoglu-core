# Yapay Zeka

## Amaç
Bu klasör, projenin yapay zeka bileşenlerini içerir. Makine öğrenimi modelleri, eğitim verileri ve AI tabanlı araçlar burada geliştirilir.

## Kullanım
AI modelleri, video düzenleme, analiz ve otomasyon süreçlerinde kullanılır. Eğitim ve test verileri ile birlikte yönetilir.

## İleride Eklenecek Dosyalar
- Eğitim veri setleri ve etiketler
- Makine öğrenimi modelleri (TensorFlow, PyTorch)
- AI API entegrasyonları
- Model performans raporları
