# Docker

## Amaç
Bu klasör, projenin konteynerleştirme konfigürasyonlarını içerir. Docker imajları ve compose dosyaları burada yönetilir.

## Kullanım
Docker araçları, geliştirme ortamlarını standardize etmek ve dağıtımı kolaylaştırmak için kullanılır.

## İleride Eklenecek Dosyalar
- Dockerfile'lar ve docker-compose.yml
- Konteyner imajları ve registry konfigürasyonları
- Ortam değişkenleri ve secrets yönetimi
- Test ve geliştirme ortamı tanımları
